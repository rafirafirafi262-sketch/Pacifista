# CCTV Monitoring Bot (Baileys + Puppeteer)

**Bot monitoring CCTV** berbasis WhatsApp menggunakan library [`@whiskeysockets/baileys`], Puppeteer (Chromium) dan Cheerio.  
Fungsi utama: memantau status monitor dari halaman status (Kuma/uptime), mengirim notifikasi via WhatsApp, serta melakukan mekanisme eskalasi dan maintenance mode.

---

## Fitur
- Scan halaman status (Kuma) dan deteksi monitor `offline` / `online`
- Notifikasi gabungan saat monitor offline / kembali online
- Anti-spam: delay, limit pesan / jam, batch send, cooldown
- Eskalasi otomatis: admin → atasan → pimpinan (berdasarkan waktu)
- Maintenance Mode: masukkan monitor ke maintenance (skip pemantauan & eskalasi) selama X waktu
- Regenerasi session otomatis jika logout / bad session
- Auto run dengan PM2 support

---

## Struktur singkat file
- `index.js` / `bot.js` — (file utama; contoh: file yang Anda kirim)
- `Session_baileys/` — folder sesi auth (autogenerated oleh Baileys)
- `package.json` — dependencies & script
- `.env` — file environment (tidak disertakan di repo; contoh di bawah)

---

## Requirement
- Node.js 18+ (direkomendasikan LTS)
- npm / yarn
- Chromium / Chrome terinstall (contoh: `/usr/bin/chromium`)
- Akses jaringan ke `KUMA_BASE_URL` (contoh: `http://172.16.100.10`)
- Port outbound untuk WhatsApp (socket)
- Akses filesystem untuk menyimpan session
